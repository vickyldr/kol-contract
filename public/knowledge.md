## 〇、新人入职流程（有人问"我该先干嘛 / 怎么开始 / 入职"时回答）

1. 先通读两份 SOP：**合同修改 SOP** 和 **支付 FAQ**（见下文，机器人也能直接答）。
2. 熟悉 **合同助手工具**（见"工具用法"）：选产品 → 粘贴红人信息一键智能填充 → 核对 → 生成合同。
3. 跟一遍完整流程：收红人信息 → 生成合同 → 发红人 → 如有修改走改约 SOP。
4. 遇到问题先在群里 **@机器人** 问（它会查 SOP/FAQ）；它答不准或涉及底线判断，再找 **TL**。
5. 任何"能不能改条款 / 金额 / 合规"的判断，**最终以 TL 确认为准**，不要自行承诺。

## 一、合同助手工具用法（autofill + 改约）

- **入口**：网页工具（部署地址问 TL）。三个页签：合同记录 / 模板 / 产品。

## 二、支付 FAQ（合作前必看：金额 / 币种 / 方式）

### 付款方式优先级
**PayPal 优先 → 不行看 Payoneer → 再确认银行转账**。银行转账前必须先查国家是否可付（《小额支付付款信息收集》表），搜不到的国家 = 不合作，不要直接答应。对公账户单独处理。

### PayPal
- 优点：到账快、限制少，**优先用**。
- 新注册账户 **21 天后**才能收款；没有 PayPal 不用强行注册，建议用亲友账号。
- 我方承担约 **2%** 手续费。
- 只收两项：**PayPal 姓名**（写错一般问题不大）+ **PayPal 账号（必须邮箱格式）**。
- 不支持 PayPal ID / paypal.me 链接 / 手机号。
- 支持币种：**美金、欧元、英镑、港币、台币、日元**；**韩币不支持** PayPal 收款。
- 可预付款、可在商家提交证明后撤回款项。

### Payoneer
- 对方没有 PayPal、且金额合适时考虑。
- **单笔少于 50 美金不能发**（2–3 个视频合并）。
- 账号一般是邮箱格式。
- **不支持追回款项，所以不能预付款**。

### 银行
- PayPal 优先，银行最后考虑；红人只能银行收款时**先别急着答应**，先查《小额支付付款信息收集》表。
- **不建议银行的情况**：单笔 < 50 美金、需要预付款、黑名单/战乱国家、对方不清楚账户能否收国际汇款。
- 每次转账约 **15 美金手续费**，小额容易不划算甚至失败。优先建议红人用亲友 PayPal 或可支付国家的亲友银行账户。
- 到账时效：我方付出后 1–3 工作日；印度本地渠道 T+2–5，回执发红人时可提醒。超 3 天可向财务索要报文。
- **收款信息要点**：
  - 全部收**英文名**，不能有特殊字符（Köksal → Koksal）；中文/繁中/本地字符名一律转英文，否则可能退回。
  - 通用信息：收款银行国家、银行名称（日本必须到支行）、IBAN/银行账号、**SWIFT/BIC（关键！8 或 11 位）**、收款人国家、**收款人全名（必须银行登记实名，可能是亲友名字，一定问清）**、收款人地址&邮编（是收款人地址，不是银行/红人地址；反洗钱要求）。
  - 美国用 Routing Number；日本必须补：银行名/支行名/账号/SWIFT/户名/地址/邮编。

### 特殊国家
- **台湾**：PayPal 可收台币/美金；**银行不能收台币，只能美金**（谈合作可从台币谈起，确认支付时推荐 PayPal；只能银行卡就改美金报价）。
- **日本**：银行信息必须补到支行等 7 项。
- **韩国**：PayPal 只收韩币；银行可韩币或美金，收信息时确认是韩币还是美金账户、能否收国际汇款。

### 黑名单 / 制裁国家
先查表格；OFAC 制裁名单上的国家（中国大陆/香港除外）**不合作**。可建议红人用亲友 PayPal，或开户地在白名单国家的银行账户。亲友账户时，签约法定名和银行户名可能不一致，注意区别（用第三方账户选项）。

### 对公支付
识别到公司账户先问：是否对公账户？是否有对私账户可收款（对私更快更方便）。若坚持对公，要求对方提供**公司资质证明 + 发票**，并发我方公司信息供其开票：
`Quvideo (Hong Kong) Limited, Suite 6503, 65/F, Central Plaza, 18 Harbour Road, Wan Chai, Hong Kong`
发票模板：银行支付用 `invoice-Quvideo.xlsx`，PayPal/Payoneer 用 `invoice-Ocean Look.xlsx`。**文件只能发红人邮箱，不能通过 Instagram 私信发。**审批时勾选"是否对公合作"并上传资质和发票。

### 常见红人问题（话术要点）
- **要家庭地址干嘛**：跨国转账必要信息（反洗钱），不信让他们自己谷歌。
- **PayPal 走"亲友/Friends&Family"**：不行。商业合作必须走 Goods & Services（合规、有凭证、有买卖保障）；我们会额外多付 2% 覆盖手续费。
- **PayPal 收款 on hold**：钱已付出（有回执，在他账号里），我们无法解决，要红人自己找客服；先排查是不是新注册账户（21 天）。
- **回执已出但没收到**：确认收款信息正确、回执已出、财务无提示；银行 1–3 工作日到账、PayPal 可能几小时延误；确认公司名（Ocean Look International Co., Limited）、是否 pending/on hold；让红人找客服查 transaction ID。
- **退款**：一般让对方先找客服退；银行退款手续费约 35 美金，过错方承担（具体找 TL）。

### 审批要点
- 收款信息必须**完整、无空**；PayPal name 要和合同/审批一致（红人只给法定名就默认 PayPal name = 法定名）。
- 备注写清楚，不能模糊（预付要说清是预付还是尾款；只付部分视频的钱也要写明）。
- **PayPal 支付的公司名称用 Ocean Look，其它都是 Quvideo**。
- 银行户名：台湾/香港/土耳其只能英文字母；日本/韩国可本国名。确认是否美元账户、能否收国际汇款。

## 三、合同修改 SOP（红人要改条款时）

红人发合同后提出改条款，按此流程，**不能自行判断/承诺/发出，必须内部确认**：

1. **整理修改点**：把红人原话、具体想改什么整理出来。
2. **发 TL 判断**：哪些能改、哪些不能改、能改的改成什么样（尽量拿到明确方向，如改成 12 个月）。
3. **先和红人同步内部判断**：正式改合同前，先把可接受方案同步红人，看能否接受。
4. **用 AI/GPT 按最小改动原则改合同**：优先替换局部词句或加限定语，不要大幅重写或删大段。
5. **改动明显标记（高亮）→ 发 TL 复核**：确认是否正确/完整/有无多改漏改。**TL 确认无误才能发红人**。
6. **发红人确认并签署**。

常见可改条款（英文为例）：
- **授权时长（最高频）**：原文 exclusive, permanent, unlimited, worldwide…；直接把 permanent 改成对应时间，有诉求 exclusive 可改 non-exclusive，可加"仅限本次活动"。
- **删除 gift card**：搜关键词 gift，可整段删。
- **甲方可随时终止**：搜 any time。
- **审批权（甲方不同意就不付钱）**：搜 approve/approval。
- **预付**：把"发布后支付"改成"初稿提交后预付 50%、发布后 50%"。
- **二改要提前通知**：在授权条款后加"任何重大修改/二次使用需提前告知乙方确认"。
